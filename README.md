Personal dotfiles


### Zsh

https://grml.org/zsh/

```
wget -O ~/.zshrc https://git.grml.org/f/grml-etc-core/etc/zsh/zshrc
wget -O ~/.zshrc.local https://raw.githubusercontent.com/dofuuz/dotfiles/main/.zshrc.local
chsh -s $(which zsh)
```
